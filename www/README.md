# 🌅 The First Life — الحياة الأولى

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **الحياة الأولى (The First Life)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on origins of creation, purpose of existence, human nature, soul, body, worldly life vs afterlife
- 6 deep reflections with Quranic evidence
- Daily habit tracker, self-assessment quiz, duas
- Trilingual: Arabic, English, French with RTL/LTR
- 3 themes (Blue, Night, Sky), Narrator, PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.
